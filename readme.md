## Instalação
Para instalar as dependências do projeto, execute:
```bash
npm install
```

## Configuração
Crie um arquivo `.env` na raiz do projeto com as seguintes variáveis:
```
DB_HOST=seu_host
DB_USER=seu_usuario
DB_PASS=sua_senha
```

## Uso
Para iniciar o servidor, execute:
```bash
npm start
```
O servidor estará disponível em `http://localhost:3000`.
